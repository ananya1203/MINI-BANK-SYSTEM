Mini Banking Simulation System

Overview
The Mini Banking Simulation System is a simple Python-based console application that simulates basic banking operations. It allows users to create an account, deposit money, withdraw money, check balance, and exit the system. This project is designed for beginners to understand core concepts of Python such as loops, functions, and conditional logic.

Features
•	- Create Account – Register a user with name, age, and initial deposit
•	- Deposit Amount – Add money to the account
•	- Withdraw Amount – Safely withdraw money with balance validation
•	- Check Balance – Display the current balance
•	- User-Friendly Menu – Simple text-based navigation
•	- Error Handling – Prevents invalid inputs and insufficient balance
•	- Runs in Jupyter Notebook – Fully compatible and error-free

Technologies / Tools Used
•	- Python 3.x
•	- Jupyter Notebook
•	- Built-in Python modules (no external libraries needed)

Steps to Install and Run the Project
1.	1. Install Python from https://www.python.org/downloads/
2.	2. Install Jupyter Notebook:
   pip install notebook
3. Open Jupyter Notebook:
   jupyter notebook
4. Create a new .ipynb file and paste the project code.
5. Run the program and follow the on-screen menu.

Instructions for Testing
3.	1. Test Account Creation – Validate user details and initial balance.
4.	2. Test Deposit – Deposit valid amounts and verify balance updates.
3. Test Withdrawal – Test valid and invalid withdrawal cases.
4. Test Balance Inquiry – Ensure accurate balance display.
5. Test Invalid Inputs – Try negative, empty, or non-numeric inputs.
